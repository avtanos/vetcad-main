## Использование

### Создание мокап данных

```bash
cd backend
python -m app.mock_data
```

### Очистка и пересоздание

Скрипт автоматически очищает старые данные перед созданием новых.

### Вход в систему

Используйте любой из созданных аккаунтов:

- **petowner** / password123 - для тестирования функционала владельца
- **veterinarian** / password123 - для тестирования функционала ветеринара
- **partner** / password123 - для тестирования функционала партнера

## Примечания

- Все пароли: `password123`
- Данные создаются заново при каждом запуске скрипта
- Старые данные удаляются перед созданием новых
- Для работы скрипта необходимо сначала запустить `init_db.py` для создания типов животных
