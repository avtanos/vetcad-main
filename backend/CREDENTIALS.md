## Использование

1. Убедитесь, что мокап данные созданы:
   ```bash
   cd backend
   python -m app.mock_data
   ```

2. Запустите бэкенд:
   ```bash
   python run.py
   ```

3. Запустите фронтенд:
   ```bash
   cd vetcard-main
   npm run dev
   ```

4. Войдите в систему с любым из тестовых аккаунтов на странице `/login`

## Примечания

- Пароли для обычных пользователей: `password123`
- Пароль для супер администратора: `admin123`
- Данные создаются заново при каждом запуске `mock_data.py`
- Старые данные удаляются перед созданием новых
- Для работы необходимо сначала запустить `init_db.py` для создания типов животных
